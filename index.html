Careers in IT

IT Support

IT Support monitors and maintains the company computer systems, installs, and configures hardware and software, and solves technical issues as they arise.

Artificial Intelligence Engineer

AI engineers develop AI-driven solutions, including chatbots, recommendation systems, and autonomous vehicles. They apply machine learning and deep learning techniques to solve complex problems.

Game Developer

A game developer is a professional who designs, creates, and develops video games. Game developers work on all aspects of game production, including conceptualizing game ideas, designing gameplay mechanics, creating graphics and visual assets, writing code, and testing and debugging games.

UX/UI Designer

User experience (UX) and user interface (UI) designers focus on creating intuitive and visually appealing interfaces for software and applications, enhancing user satisfaction and engagement.

Application Developer

An app developer is a professional who designs, creates, and develops applications (apps) for various computing platforms, such as mobile devices, tablets, computers, and other digital devices.

Website Developer

Web developers design and create websites and web applications. They work with various programming languages, frameworks, and design tools to build interactive and visually appealing web experiences.

Google, Apple, Meta, Amazon, Windows, and Tesla are in the top richest companies in the world.

BS in SoftWare Development

The bachelor's degree in software development prepares students with the programming and development skills they need for a career in software development with particular emphasis in web development.

BS in Information Technology

The bachelor's degree in information technology helps students prepare for careers in software development, web development, and other technology-related fields. Students gain specific technical skills by selecting from a variety of options, including computer support, programming, web and database development, and system administration.
